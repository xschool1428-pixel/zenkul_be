# School SaaS API

Multi-school education platform API built with **Node.js**, **Express**, **MongoDB**, **Socket.io**, **JWT**, and **Razorpay**.

## Quick start

```bash
cd school-saas-api
cp .env.example .env
# Edit .env with MongoDB URI, JWT secrets, Razorpay keys

npm install
npm run dev
# Optional demo data: npm run seed
```
